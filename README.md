## ¿Por qué del proyecto?
Para ordenar un poco mis gastos y saber a dónde se va mi dinero.

## Stack
Las tecnologías utilizadas fueron las siguientes:

    - MongoDB
    - Express.js
    - Angular v6
    - Node.js
